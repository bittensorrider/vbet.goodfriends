# vbet.goodfriends

Vietnamese Casino Bet - **GoodFriends**

> Client

## Prerequisites

```yaml
node.js: 20.20.2
npm: 10.8.2

next.js: 15.3.1
```

## Getting Started

```env
NEXT_PUBLIC_API_URL=http://82.76.143.138:16610
```

```bash
npm i --legacy-peer-deps
```

First, run the development server:

```bash
# run hot-reloading dev server
npm run dev

# build
npm run build
# start
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How To Deploy

Use **Vercel** to deploy.
